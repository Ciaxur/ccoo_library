# ccoo library
oomar's c++ library. this is a library that contains repetative containers that would make sense to
be in std, but is not and no need to bloat a simple project with boost libraries.

## Inspiration
Data structures like LinkedLists should have functionality of removing elements from within the list
itself, not just poping either ends. I find myself re-implementing this over and over again.
